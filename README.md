# TempleId


<p> TempleId adalah aplikasi iOS daftar candi candi yang ada di indonesia

Aplikasi ini dibuat untuk memenuhi syarat kelulusan dari kelas dicoding "Belajar membuat aplkiasi iOS untuk pemula". </p>


## Requirements
- macOS 11.5.1
- Xcode 12.5.1
- iOS 14.0
